Make a Github Blog

新建用户名.github.io仓库

![Macdo2025-12-12 17.40.49](Make a Github Blog/Macdo2025-12-12 17.40.49.png)

打开终端执行

npm install -g hexo-cli获得Hexo插件

用 hexo 创建博客

hexo init myblog

安装 hexo 的 git 部署插件

npm install hexo-deployer-git –save

cd ~/Desktop/myblog

把 Butterfly 主题克隆到 themes/butterfly 目录

git clone https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

找个编辑配置Hexo的_config.yml

nano _config.yml

添加下面的部分在最下面

更改 theme主题

deploy:
type: git
repo:
branch: main

然后ctrl+O Enter ctrl+X

可选：卸载旧的 ejs/jade 渲染器，避免冲突

npm uninstall hexo-renderer-ejs hexo-renderer-jade

安装 Butterfly 要用的渲染器

npm install hexo-renderer-pug hexo-renderer-stylus –save

卸载旧的图片适配器

npm uninstall hexo-asset-image

安装新的图片适配器

npm install hexo-asset-image –save

例子

MacGesture2-Publish
├── apppicker.jpg
├── logo.jpg
└── rules.jpg
MacGesture2-Publish.md

如果有一个外部文章md

连同它的图片文件移动到/Users/jimmy/Desktop/myblog/source/_posts

然后打开终端回到myblog
